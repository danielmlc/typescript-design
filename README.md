#### typescrit知识小结

`extends` 关键字 用在类继承中，继承类如果是实体类必须在构造函数中执行super()方法执行父类的构造函数；并且必须实现父类的虚方法。

`implements` 关键字 用在继承接口或者类，继承时必须实现接口或类中定义的变量或方法。
